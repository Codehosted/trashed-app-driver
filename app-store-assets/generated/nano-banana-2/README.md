# Nano Banana 2 store assets

Generated on 2026-07-19 with Google `gemini-3.1-flash-image` (Nano Banana 2).

- `route-photo-01-access.jpg`: synthetic driveway access-condition photo.
- `route-photo-02-placement.jpg`: same synthetic site after roll-off placement.
- `route-photo-03-complete-detail.jpg`: deterministic crop of the placement photo for detail coverage.
- `store-background-9x16.jpg`: abstract background plate for product screenshot composition.

All customer, site, and job content is synthetic. The route photos were visually inspected at original resolution for malformed geometry, duplicated objects, stray text/logos, people, faces, plates, and watermarks. Product UI is composited separately so AI generation cannot alter interface text, icons, maps, or controls.
