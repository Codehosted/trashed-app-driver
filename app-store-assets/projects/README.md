## App listing

### App name
Trashed Projects

### App identity
`app.trashed.projects`

### Subtitle / short description
A reuse marketplace for Builders and Supporters.

### Full description
Trashed Projects helps Builders and Supporters find and share usable materials, fixtures, equipment, and project leftovers before they become waste. Listings focus on mass-market reuse: items are free or up to 50% off retail, with points awarded for value, generosity, and keeping useful materials in circulation.

### Key features
- Browse project materials, supplies, and reusable goods.
- Find free or up to 50% off retail items from local Builders and Supporters.
- Earn points for useful listings, generous offers, and reuse activity.
- Open the Trashed Projects web marketplace in a dedicated mobile shell.

## Privacy and data-safety copy

Trashed Projects is configured as a web-shell marketplace app and does not request location permission. It does not inherit the Trashed Driver foreground location, background location, or foreground-service location permissions. If optional pickup logistics are added later, location access should be introduced as a separate opt-in feature with updated store copy, platform prompts, and tests.

Expected data is limited to account, listing, marketplace interaction, device/network, and support information handled by the Trashed web experience. This draft does not include camera, contacts, microphone, health, calendar, foreground location, or background location permissions.
